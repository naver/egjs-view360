Initial pitch(X-axis rotation) value.
Look up if initialPitch > 0 and down if initialPitch < 0.

A value between `-90 and 90` must be used.
Values beyond that range are clamped into the range.

  projectionOptions={{
    src: "/pano/equirect/veste.jpg"

  showExampleCode />
