Source URL to Panorama image / video.
By default, it uses `equirectangular` projection, but you can change it with [projectionType](/docs/options/PanoViewer/source/projectionType/equirectangular)
<PanoViewer
  src="/egjs-view360/pano/equirect/veste.jpg"
  showExampleCode />

It also supports an array of cubemap URLs.
The order is **"RLUDFB(+X, -X, +Y, -Y, +Z, -Z)"** by default, but you can change it with `cubemapOrder`
