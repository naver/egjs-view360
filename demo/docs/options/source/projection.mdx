[Projection](/docs/api/Projection/Class/Projection) for displaying panorama image/video.
A list of the available projections and how to use each projection can be found in the [Projection Guide](../../projections).

## Example
```js
import View360, { EquirectProjection } from "@egjs/view360";

const viewer = new View360("#el_id", {
  projection: new EquirectProjection({
    src: "URL_TO_PANO_IMAGE_OR_VIDEO"
  })
})
```
