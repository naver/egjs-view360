최초 yaw(Y축 회전)값을 설정합니다.

0~180 범위 내에서
initialYaw > 0 일 경우 왼쪽을, initialYaw < 0일 경우 오른쪽을 바라봅니다.

숫자 범위에 제약은 없으나, 실제 `yaw`를 설정할 때는 0~360 범위 내에서 저장됩니다.
예를 들어, `-60`의 경우 `300`을, `420`의 경우 `60`을 실제 값으로 사용합니다.

## 예시
<View360
  projectionOptions={{
    src: "/pano/equirect/veste.jpg"
  }}
  initialYaw={60}
  license="veste"
  showExampleCode />

<View360
  projectionOptions={{
    src: "/pano/equirect/veste.jpg"
  }}
  initialYaw={-60}
  license="veste"
  showExampleCode />
