`true`일 경우 자이로스코프 입력의 roll(z축 회전)값을 무시합니다.
:::caution
이 값을 `false`로 설정할 경우 카메라 범위 제약을 무시합니다.
[yawRange](/docs/options/camera/yawRange), [pitchRange](/docs/options/camera/pitchRange)와 같은 값은 무시되며, [CylinderProjection](/docs/projections/cylindrical) 사용시에도 범위를 벗어날 수 있습니다.
:::

## 예시
<View360
  projectionOptions={{
    src: "/pano/equirect/veste.jpg"
  }}
  gyro={{
    ignoreRoll: true
  }}
  license="veste"
  showExampleCode />

<View360
  projectionOptions={{
    src: "/pano/equirect/veste.jpg"
  }}
  gyro={{
    ignoreRoll: false
  }}
  license="veste"
  showExampleCode />
