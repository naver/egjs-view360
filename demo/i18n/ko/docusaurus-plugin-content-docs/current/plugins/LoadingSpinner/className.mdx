디폴트 클래스 이름들을 오버라이드합니다.
컨트롤바 엘리먼트에 별도의 이름을 설정하고자 할 때 사용하실 수 있습니다.

## 예시
아래는 컨테이너 엘리먼트(`.view360-spinner`)의 클래스를 바꾸는 예시입니다.
```js
import { LoadingSpinner } from "@egjs/view360";

new LoadingSpinner({
  className: {
    CONTAINER: "new-container-classname"
  }
})
```
