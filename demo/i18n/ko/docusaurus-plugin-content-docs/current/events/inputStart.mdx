```ts
import { EVENTS, InputStartEvent } from "@egjs/view360";

// typescript 코드입니다.
const onInputStart = (evt: InputStartEvent) => {};

view360.on(EVENTS.INPUT_START, onInputStart);
```

사용자 입력 시작 시점에 발생하는 이벤트.

## 프로퍼티
### srcEvent: MouseEvent | TouchEvent | KeyboardEvent
사용자 입력의 [Event](https://developer.mozilla.org/en-US/docs/Web/API/Event) 인스턴스.

### inputType: "rotate" | "zoom"
사용자 입력의 종류

### isTouch: boolean
사용자 입력이 터치를 통한 것이었는지 여부

### isKeyboard: boolean
사용자 입력이 키보드를 통한 것이었는지 여부

:::info
예제에서 확인 가능한 이벤트 목록은 다음과 같습니다.
- "inputStart", "inputEnd", "staticClick"
:::
