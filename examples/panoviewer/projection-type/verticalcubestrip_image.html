Customized Cubemap

Cubemap can have different layout, different order, diffent rotation

projection: CUBEMAP

cubemapConfig:
{tileConfig: {flipHorizontal: true, rotation: 0}}

image: 

var PanoViewer = eg.view360.PanoViewer;
var panoViewer = new PanoViewer(container, {
    projectionType: "cubemap",
    cubemapConfig: {
        tileConfig: {flipHorizontal: true, rotation: 0}
    },
    image: "../../img/cube.jpg"
});
